Tipos de Input type


Imput type Descripcion Ejemplos
Input type text Crea un campo de texto de una sola linea
Input type password Crea un campo para la contraseña
Input type submit Envia el formulario al Backend
Input type reset Crea un boton para limpiar el contenido de los campos
Input type radio Crea un boton. Los botones de radio permiten al usuario seleccionar SOLO UNA de un número limitado de opciones
Input type checkbox Crea una casilla de verificacion. Es parecido al TYPE RADIO, pero permite varias mas

Input type button Crea un Boton
Input type color Se utiliza para campos de entrada que deben contener un color.
Imput type date Crea un campo de para agregar fechas
Input type Datetime-local especifica un campo de entrada de fecha y hora, sin zona horaria.
Input type Email Crea un campo para agragar un correo electronico
Input type file Define un campo de selección de archivo y un botón "Examinar" para la carga de archivos.
Input type Month Permite al usuario seleccionar un mes y un año.
Input type Number Define un campo de entrada numérico .
Input Range define un control para ingresar un número cuyo valor exacto no es importante (como un control deslizante). Rango por defecto es de 0 a 100. Sin embargo, se puede establecer restricciones en lo que los números son aceptados con el min, maxy stepatributos
Input type search Crea un campo de busqueda
Input type Tel Crea un campo para numeros telefonicos
Input type time Permite al usuario seleccionar una hora (sin zona horaria).
Input type URL Se utiliza para campos de entrada que deben contener una dirección URL.
Input type Week Permite al usuario seleccionar una semana y un año.



Input Restrictions

Atributos Descripcion
checkend Especifica que se debe preseleccionar un campo de entrada cuando se carga la página (para type = "checkbox" o type = "radio")
disabled Especifica que un campo de entrada debe estar deshabilitado
max Especifica el valor máximo para un campo de entrada
maxlength Especifica el número máximo de caracteres para un campo de entrada
min Especifica el valor mínimo para un campo de entrada
pattern Especifica una expresión regular para comparar el valor de entrada
readonly Especifica que un campo de entrada es de solo lectura (no se puede cambiar)
required Especifica que se requiere un campo de entrada (debe completarse)
size Especifica el ancho (en caracteres) de un campo de entrada
step Especifica los intervalos de números legales para un campo de entrada
value Especifica el valor predeterminado para un campo de entrada
