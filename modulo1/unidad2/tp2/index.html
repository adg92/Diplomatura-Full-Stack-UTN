Tipos de input

Input Type Descripción Ejemplo
Input Type Text Crea un campo de texto de una sola línea
Input Type Password Crea un campo de contraseña
Input Type Submit Crea un botón para enviar el formulario para su procesamiento en el backend
Input Type Reset Crea un botón para limpiar el contenido de los campos del formulario
Input Type Radio Crea un botón de radio. Los botones de radio permiten al usuario seleccionar solo una de un número limitado de opciones    
   
Input Type Checkbox Crea una casilla de verificación (checkbox). Permite seleccionar una o más opciones de una cantidad limitada de opciones        
Input Type Button Crea un botón
Input Type Color Se usa para campos de entrada que contienen colores    
Input Type Date Se utiliza para campos de entrada que contienen fechas    
Input Type Datetime-local Crea un campo de entrada de fecha y hora sin zona horaria    
Input Type Email Crea un campo de entrada de email    
Input Type File Crea un campo para seleccionar y cargar archivos    
Input Type Month Crea campos de entrada de mes y año    
Input Type Number Crea un campo de entrada de solo números    
Input Type Range Define un control para ingresar un número cuyo valor exacto no es importante. El rango predeterminado es de 0 a 100. Sin embargo, se pueden establecer restricciones sobre los números que se aceptan con los atributos mínimo(min), máximo(max) y de paso(step).    
Input Type Search Crea un campo de búsqueda    
Input Type Tel Crea un campo para introducir un número de teléfono    
Input Type Time Crea un campo para seleccionar una hora    
Input Type Url Crea un campo para introducir una URL    
Input Type Week Crea un campo para seleccionar semana y año    

Restricciones de input

Atributo Descripción
checked Se usa en input type checkbox o type radio. El campo con este atributo aparecerá seleccionado automáticamente al cargar el formulario
disabled El campo que contenga este atributo estará desactivado
max Especifica el valor máximo para un campo de entrada
maxlength Especifica el número máximo de caracteres para un campo de entrada
min Especifíca el valor mínimo para un campo de entrada
pattern Permite usar expresiones regulares para comparar el valor de un campo de entrada
readonly Especifica un campo de entrada como solo lectura (no puede ser modificado)
required El campo con este atributo es requerido (debe ser llenado)
size Especifica el ancho (en caracteres) de un campo de entrada
step Especifica los intervalos de números legales para un campo de entrada
value Especifica el valor predeterminado para un campo de entrada
